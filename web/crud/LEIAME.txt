Instruções:
1. Instale MySQL Server, MySQL Workbench, Nodejs e VS Code (ou VS Codium)
2. Inicialize o serviço do MySQL 
3. Abra o MySQL Workbench
4. Clique em 'Data Import/Restore' na aba 'Administration'
5. Selecione 'Import from Self-Contained File'
6. Selecione o arquivo 'mercadao.sql' e clique em 'Start Import'
7. Abra o VSCode ou VSCodium
8. Instale a extensão Live Server e NPM
9. Abra um terminal e digite npm init, clicando enter até o terminar o setup inicial
10. Digite npm install express cors mysql2
11. Instale Live Server caso não esteja já instalado.
12. Abra o diretório contendo os arquivos do projeto
13. Abra um terminal e digite 'node server.js'
14. Clique no arquivo 'index.html'
15. Clique no ícone do Live Server e a página será aberta em seu navegador padrão
