Instruções:
1. Inicialize o serviço do MySQL 
2. Abra o MySQL Workbench
3. Clique em import
4. Selecione o arquivo dump
5. Realize a importação do banco de dados
6. Abra o VS Code
7. Abra a pasta contendo os arquivos do projeto
8. Abra um terminal e digite 'node server.js'
9. Clique no arquivo 'index.html'
10. Clique no ícone do Live Server
